### Prerequisites

- Node.js 18+ and npm

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/meditation-timer.git
   cd meditation-timer
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   ```

4. **Open [http://localhost:3000](http://localhost:3000) in your browser.**
